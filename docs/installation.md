# Installation

## NPM

```bash
npm install react-components
```

## Yarn

```bash
yarn add react-components
```

## PNPM

```bash
pnpm add react-components
```

## Requirements

- React >= 19.0.8
- TypeScript (recommended)

## Getting Started

After installation, import the components you need:

```tsx
import { DialogProvider, useDialog } from 'react-components';
```

For TypeScript projects, all type definitions are included automatically.
